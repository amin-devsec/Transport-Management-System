# Transport Management System

A Transport Management System developed in **C++** as a semester project. The application demonstrates object-oriented programming concepts and provides basic transport management functionalities.

## Author

**Amin Ullah**
Cyber Security Student (4th Semester)

## Features

* Vehicle management
* Passenger management
* Route management
* Booking system 
* File handling 
* Object-Oriented Programming concepts


## Requirements

* C++11 or later
* Visual Studio Code (or any C++ IDE)
* GCC, MinGW, or any C++11-compatible compiler

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Amin-CyberWarrior/Transport-Management-System.git
   ```

2. Open the project in Visual Studio Code.

3. Compile using a C++11 (or newer) compiler.

Example:

### Compile

```bash
g++ -std=c++11 src/TransportManagmentSystem.cpp -o transport
```

### Run

**Linux/macOS**
```bash
./transport
```

**Windows**
```bash
transport.exe
```
